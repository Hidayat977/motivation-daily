🌐 STEP 4: Deploy with GitHub Pages (Free Hosting)

  1. Go to your repo → Settings → "Pages" (or "Pages & Deployment")

  2. Under "Source", choose:

    • Branch: main

    • Folder: / (root)

  3. Click Save

  4. GitHub will give you a link like:
    https://yourusername.github.io/daily-motivation/
